# pycloud
this is a little tool i made for myself, it uses telegram and python to make something like a cloud file hosting service BUT you get unlimited storage

## Installation
get ***python**(most versions are supported, latest one is reccomended)*

install requirements yourself or run **install.bat** to get the needed libraries automatically.

run pycloud.py through the command prompt using pycloud.py while in the file directory(use cd C:/whatever)
